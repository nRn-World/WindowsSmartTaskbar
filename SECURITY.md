# Security Policy

## Supported Versions

Currently, only the latest version of Windows Smart Taskbar is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Windows Smart Taskbar seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, report it privately.

Please report security vulnerabilities to: **bynrnworld@gmail.com**

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary.
